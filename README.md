# Tip Calculator

A simple, user-friendly Tip Calculator web application that helps you easily calculate the tip amount and the final amount each person needs to pay. Built using HTML, CSS, and JavaScript.

## 🚀 Features

- **Bill Splitter:** Enter the total bill amount, the desired tip percentage, and the number of people to easily split the cost.
- **Accurate Calculation:** Automatically calculates the total tip and the exact amount each person owes.
- **Input Validation:** Prompts the user to enter valid numbers if the inputs are left empty or zero.
- **Clean Interface:** A straightforward card layout for quick calculation.

## 🛠️ Technologies Used

- **HTML5:** Structures the calculator interface.
- **CSS3:** Adds styling to the calculator card and elements (`style.css`).
- **JavaScript:** Performs the mathematical calculations and updates the DOM dynamically (`main.js`).

## 📁 Project Structure

```text
Tip calculator/
├── index.html   # Main HTML layout for the calculator
├── style.css    # CSS file for the design
├── main.js      # JavaScript file containing the calculation logic
└── README.md    # Project documentation
```

## 💻 How to Use

1. Navigate to the `Tip calculator` folder.
2. Open the `index.html` file in your preferred web browser.
3. **Enter Persons:** Enter the total number of people splitting the bill.
4. **Enter Amount:** Enter the total bill amount.
5. **Enter Tip:** Enter the tip percentage you wish to leave.
6. Click the **"Calculate"** button to see the total tip and the amount per person!
